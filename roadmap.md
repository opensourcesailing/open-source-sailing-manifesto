# ⚓ Open Source Sailing Roadmap

## Building the future of open marine technology

This roadmap describes the long-term vision and development path of Open Source Sailing.

The goal is not only to create products, but to build a global open-source marine technology ecosystem.

---

# Phase 1 — Foundation (2026)

## Establish the Open Source Sailing ecosystem

Goals:

- Create project identity and documentation
- Build website and online presence
- Develop GitHub structure
- Grow YouTube and community channels
- Publish the Open Source Sailing Manifesto
- Create educational content

Focus:

⚓ Vision  
📚 Documentation  
🤝 Community building

---

# Phase 2 — Exploration & Prototypes (2026–2027)

## Build and test open marine solutions

Projects:

- ESP32 marine electronics
- NMEA0183 gateways
- NMEA2000 experiments
- OpenCPN integrations
- Signal K implementations
- SDR marine communication experiments
- Raspberry Pi and Linux onboard systems

Focus:

🔧 Prototyping  
🧪 Testing  
📡 Integration

---

# Phase 3 — Open Marine Platform (2027–2028)

## Develop a complete open onboard ecosystem

Goals:

- Open Source Sailing Marine Computer
- Standardized hardware modules
- Installation guides
- Software images
- Community testing

Potential components:

- Linux LTS
- OpenCPN
- Signal K
- MQTT
- Grafana
- Home Assistant
- AIS
- SDR
- Sensor networks

Focus:

💻 Software platform  
🔌 Hardware ecosystem

---

# Phase 4 — Community Growth (2028–2030)

## Build a global network of contributors

Goals:

- Expand documentation
- Create contributor programs
- Organize online events
- Collaborate with open-source projects
- Support educational initiatives

Focus:

🌍 Global community  
🤝 Collaboration

---

# Phase 5 — Sustainable Open Marine Ecosystem (2030+)

## Long-term vision

A mature ecosystem where sailors can:

- Build their own systems
- Repair existing equipment
- Share improvements
- Access open marine technology worldwide

Potential developments:

- Open hardware products
- Educational academy
- Research collaborations
- Community-driven standards
- Sustainable marine technology solutions

---

# Our Direction

Open Source Sailing is a long-term journey.

We believe the future of sailing technology should be:

⚓ Open  
🔧 Repairable  
🌍 Sustainable  
🤝 Collaborative  
🚢 Accessible
